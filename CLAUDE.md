# Claude活用ガイド

このツールで生成したコンテキストを使って、Claudeに優秀なインタビュアーになってもらうためのガイドです。

## 🤖 なぜClaude？

Claudeはインタビューに特に適した特徴があります：

- **自然な会話フロー**: 人間らしい対話で被験者がリラックスできる
- **深掘り質問**: 表面的でない、本質を探る質問が得意
- **文脈保持**: 長時間の会話でも一貫して文脈を理解
- **柔軟な対応**: 予想外の回答にも臨機応変に反応
- **共感的姿勢**: 被験者の気持ちに寄り添った質問ができる

## 🎯 基本的な使い方

### 1. コンテキストの設定
1. このツールで自分用のコンテキストを生成
2. [Claude.ai](https://claude.ai) で新しいチャットを開始
3. 生成されたプロンプトをペースト
4. 「インタビュアーとして質問してください」と指示

### 2. インタビュー開始
```
あなた: プロダクトのユーザーインタビューをお願いします。
新しいタスク管理アプリについて30分程度で聞いてください。

Claude: 承知しました！リラックスしてお話しください。
まず、普段どのような方法でタスク管理をされていますか？
```

## 📊 活用シーン別のコツ

### プロダクト開発
**目的**: ユーザーニーズや使用体験の理解

**Claudeへの指示例**:
```
[あなたのコンテキスト]

上記の人物として、新しい家計簿アプリについて
ユーザーインタビューを実施してください。

以下の点を重点的に聞いてください：
- 現在の家計管理方法と不満点
- アプリに求める機能
- 使いやすさの優先順位
- 料金に対する考え方

30分程度の想定で、自然な会話で進めてください。
```

### マーケティングリサーチ
**目的**: ターゲット層の行動や価値観の理解

**Claudeへの指示例**:
```
[あなたのコンテキスト]

上記の人物として、Z世代向けファッションブランドについて
マーケティングリサーチのインタビューを行ってください。

聞きたいポイント：
- ファッション選びの基準
- 情報収集の方法
- 購入決定の要因
- ブランドへの期待

被験者がリラックスして本音を話せるよう心がけてください。
```

### UXリサーチ
**目的**: UI/UXの課題や改善点の発見

**Claudeへの指示例**:
```
[あなたのコンテキスト]

上記の人物として、ECサイトのUXについて
ユーザーインタビューを実施してください。

調査したい内容：
- サイト利用時の体験と感情
- 困った場面や迷った場面
- 期待と現実のギャップ
- 競合サービスとの比較

具体的なエピソードを引き出すよう質問してください。
```

## 🎨 インタビューテクニック

### 1. アイスブレイク
```
Claude: お時間をいただき、ありがとうございます！
緊張せずに、普段通りにお話しください。
まず、〇〇について普段どのように感じていらっしゃいますか？
```

### 2. 深掘り質問
```
Claude: なるほど、それは興味深いですね。
その時どのような気持ちでしたか？
もう少し詳しく教えていただけますか？
```

### 3. 具体化
```
Claude: 「使いにくい」とおっしゃいましたが、
具体的にはどの部分でそう感じましたか？
その時の状況を詳しく聞かせてください。
```

### 4. 仮説検証
```
Claude: もしかすると〇〇ということでしょうか？
または△△のような状況だったのでしょうか？
```

## 💡 Claudeを効果的に使うコツ

### インタビュー前の準備
- **明確な目的**: 何を知りたいのかをClaude に伝える
- **時間設定**: 想定時間を伝えて時間管理してもらう
- **重要ポイント**: 特に聞きたいことを優先順位付けして伝える

### インタビュー中の調整
```
# 方向修正したい時
あなた: もう少し〇〇について深掘りしてください

# 時間調整したい時  
あなた: 残り10分です、重要なポイントに絞ってください

# 質問追加したい時
あなた: △△についても聞いてもらえますか？
```

### 複数の視点を活用
```
# 同じテーマを異なるペルソナで
1回目: 20代女性エンジニアの視点でインタビュー
2回目: 40代男性マネージャーの視点でインタビュー
3回目: 60代主婦の視点でインタビュー
```

## 🔧 実践的なワークフロー

### 1. 事前準備（5分）
```
1. リサーチ目標を整理
2. 聞きたいポイントをリストアップ  
3. 適切なペルソナを選択
4. Claudeにコンテキストと目的を伝達
```

### 2. インタビュー実施（20-45分）
```
1. アイスブレイクで緊張を和らげる
2. メインテーマについて幅広く質問
3. 興味深い回答を深掘り
4. 重要ポイントの確認
5. 最後にまとめと感謝
```

### 3. 振り返り（5分）
```
あなた: 今のインタビューの重要な発見点をまとめてください

Claude: 本日のインタビューから、以下の重要な発見がありました：
1. 〇〇に対する強いニーズ
2. △△での課題が明確
3. ××への期待値が予想より高い
...
```

## 🚨 注意点とベストプラクティス

### Do's（推奨）
- ✅ 明確な目的設定
- ✅ 自然な会話フロー
- ✅ オープンクエスチョンの活用
- ✅ 被験者の感情に配慮
- ✅ 具体的なエピソードを引き出す

### Don'ts（避けるべき）
- ❌ 誘導的な質問
- ❌ 複数のことを一度に聞く
- ❌ 専門用語の乱用
- ❌ 時間オーバー
- ❌ 批判的な態度

## 📈 成果を最大化する方法

### 1. 複数セッションで多角的に
```
セッション1: 現状の課題理解
セッション2: 理想の解決策探索  
セッション3: 具体的な要件定義
```

### 2. 異なるペルソナで比較
```
同じトピックを：
- 初心者ユーザーの視点
- エキスパートユーザーの視点
- 潜在ユーザーの視点
でそれぞれインタビュー
```

### 3. 結果の構造化
```
あなた: インタビュー結果を以下の形式でまとめてください：
- 主要な発見
- ユーザーの課題  
- ニーズと要望
- 改善提案
- 次のアクションプラン
```

## 🌟 他のAIとの使い分け

### Claudeが特に優秀な場面
- 深い共感が必要なセンシティブなトピック
- 長時間の一貫した会話
- 複雑な文脈理解が必要な調査
- クリエイティブな発想が必要な場合

### 他のAIを検討する場面
- 大量データの定量分析が必要
- 特定分野の専門知識が重要
- 短時間で多くの質問を処理したい場合
